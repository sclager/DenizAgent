# 🚀 Deniz Agent v1.1

İlk kullanılabilir sürüm.

Bu sürüm ile birlikte Deniz Agent, yalnızca bir otomasyon scripti olmaktan çıkarılarak modern masaüstü uygulamasına dönüştürüldü.

---

## ✨ Yeni Özellikler

### 🖥 Modern Desktop UI
- PySide6 tabanlı modern arayüz
- Ana menü
- İşlem bazlı ekranlar

### 📄 Talep Dokümanı
- Şirket şablonundan Word dokümanı oluşturma
- Talep bilgilerinin otomatik doldurulması
- Taslak oluşturma desteği

### 📑 Mevcut Dokümandan Devam
- Hazırlanmış Word dokümanını tekrar okuyabilme
- Düzenlenmiş dokümandan SharePoint kaydı oluşturabilme

### 📋 SharePoint Otomasyonu
- Form alanlarının otomatik doldurulması
- Kişi seçim desteği
- Dropdown yönetimi
- Tarih alanlarının otomatik doldurulması
- Kaydet (Save) otomasyonu

### 🤝 Toplantı Kaydı
- Dokümansız toplantı kaydı oluşturma
- Varsayılan alanların otomatik doldurulması

### ⚡ Hızlı Erişim
- Sistem tepsisi (Tray) desteği
- Global **Ctrl + Shift + D** kısayolu
- Hızlı işlem menüsü

### 🎨 Kullanıcı Deneyimi
- Progress Bar
- Uygulama ikonu
- Daha düzenli iş akışı

---

## 🛠 Teknolojiler

- Python
- PySide6
- Playwright
- python-docx
